Null Operators:
  • value = Object?.Property (Null Conditional) - If Object is null, assigns null to value. Otherwise assigns Property to value. Prevents Null Reference exception if Object is null.
  • value = Array?[1] (Second Null Conditional) - Returns value at index 1 of the array if it exists, otherwise returns null. Prevents exception if index doesn't exist
  • value == Object ?? (Null Coalescing Operator) - Returns Object unless it is null, then it returns 0. Allows setting a default if Object is null.
  • myDelegate?.Invoke(args) (Null-conditional Delegate Invocation) - Invokes the delegate if not null.
