Loop Syntax

For
for (int x=0; x < myArr.Length; x++) {
 Console.WriteLine(myArr[x]);
}

For Each
foreach (int a in myArr) {
 total += a;
}

While
while (x < 10) {
 Console.WriteLine("still");
 x++;
}

Do While
do {
 Console.WriteLine("still");
 x++;
} while (x < 10);

Linq Commands

Must include "using System.Linq;" when doing any Linq commands.

  • .Select(x=>x)
  • .Where(x=>x)
  • .OrderBy(x=>x)
  • .ThenBy(y=>y)
  • .OrderByDescending(x=>x)
  • .ThenByDescending(y=>y)
  • .ToArray()
  • .ToDictionary()
  • .ToList()
  • .Reverse()
  • .Take(n)
  • .Max()
  • .Min()
