Loop Syntax

For
for (int x=0; x < myArr.Length; x++) {
 Console.WriteLine(myArr[x]);
}

For Each
foreach (int a in myArr) {
 total += a;
}

While
while (x < 10) {
 Console.WriteLine("still");
 x++;
}

Do While
do {
 Console.WriteLine("still");
 x++;
} while (x < 10);

Linq Commands

Must include "using System.Linq;" when doing any Linq commands.

  • .Select
  • .Where
  • .OrderBy
  • .ThenBy
  • .OrderByDescending
  • .ThenByDescending
  • .ToArray
  • .ToDictionary
  • .ToList
  • .Reverse
